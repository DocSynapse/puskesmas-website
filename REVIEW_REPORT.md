### TypeScript `verbatimModuleSyntax` Note

The 2026 TypeScript best practice recommends enabling `verbatimModuleSyntax: true` in
`tsconfig.app.json`. This requires all type-only imports to use `import type` syntax.
The current codebase uses shadcn/ui components which mix value and type imports naturally,
so this change is safe to implement but requires careful verification per file.

**Recommendation:** Add `"verbatimModuleSyntax": true` to `tsconfig.app.json` in the next
sprint, after confirming all imports in `src/` use `import type` for type-only symbols.
